# Zahra Typeface Font Files

Please add your Zahra Typeface font files to this directory.

## Required Font Files

Add the following font files to this directory:

- `Zahra-Regular.otf` or `Zahra-Regular.ttf` (weight: 400)
- `Zahra-Medium.otf` or `Zahra-Medium.ttf` (weight: 500)
- `Zahra-Semibold.otf` or `Zahra-Semibold.ttf` (weight: 600)

## Where to Get Zahra Typeface

You can purchase and download Zahra Typeface from:
- TheHungryJPEG: https://thehungryjpeg.com/product/121612-zahra-typeface
- Creative Fabrica: https://www.creativefabrica.com/product/zahra/

## Installation Steps

1. Purchase and download the Zahra Typeface font package
2. Extract the font files
3. Copy the font files (`.otf` or `.ttf` format) to this `public/fonts/` directory
4. Ensure the files are named exactly as specified above
5. The font will automatically be loaded when you run the application

## Notes

- The font is configured to use OpenType (`.otf`) format first, then TrueType (`.ttf`) as fallback
- If your font files have different names, update the `@font-face` declarations in `app/globals.css`
- Make sure you have the appropriate license for web use
